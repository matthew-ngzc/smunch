First time running, need to install dependencies
Create virtual env for dependencies (run from the telegram folder)
python -m venv venv

Start virtual env
venv\Scripts\activate

Install dependencies
pip install -r requirements.txt



Everytime running the bot from terminal

# Windows
venv\Scripts\activate

to stop 
deactivate

# macOS/Linux
source venv/bin/activate


Then to run the bot
python tele_bot.py


If adding new dependency, delete the requirements.txt, then switch to command prompt and regenerate dependencies file using 
pip freeze > requirements.txt
